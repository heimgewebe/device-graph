# device-graph

device-graph kartographiert Geräte, Netzwerke, Rollen, Beziehungen, Vertrauenszonen und geplante Migrationen im erweiterten Geräteumfeld des Nutzers. Das Repo ist kein Konfigurations-Repo einzelner Hosts, sondern das geräteübergreifende Modell des Gesamtbestands und seiner Netzwerkanbindung.
